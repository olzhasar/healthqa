Source code for [backpainfree.org](https://backpainfree.org) - a Q&A platform similar to StackOverFlow, which is designed specifically for people with back pain problems. Users can ask questions, post answers and comments, vote on other posts.

## Used technologies

- [Flask](https://flask.palletsprojects.com/)
- [htmx](https://htmx.org/)
- [alpine.js](https://alpinejs.dev/)
- [tailwindcss](https://tailwindcss.com/)
- [Jinja2](https://jinja.palletsprojects.com/)
- [PostgreSQL](https://www.postgresql.org/)
- [Redis](https://redis.io/)
- [MeiliSearch](https://www.meilisearch.com/)
- [Docker](https://www.docker.com/)
